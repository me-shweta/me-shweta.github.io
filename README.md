# Shweta's Portfolio - UI/UX Designer & Researcher

A modern, responsive portfolio website showcasing UI/UX design work, built with React, Tailwind CSS, and Framer Motion.

## 🚀 Quick Start

### For First Time Setup:

1. **Install Node.js** (if not already installed)
   - Download from: https://nodejs.org/
   - Version 18 or higher required

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start Development Server**
   ```bash
   npm run dev
   ```

4. **Open in Browser**
   - Visit: http://localhost:3000

That's it! 🎉

## 🎨 Customization

### Update Personal Information
- **Hero Section**: `src/components/Hero.jsx`
- **About Section**: `src/components/About.jsx`
- **Portfolio Projects**: `src/components/Portfolio.jsx`
- **Resume**: `src/components/Resume.jsx`
- **Achievements**: `src/components/Achievements.jsx`

### Change Colors
Edit `tailwind.config.js`:
```js
colors: {
  primary: '#e87532',    // Your brand color
  heading: '#0f2943',
  text: '#0a0f14',
}
```

### Add Images
1. Place images in `public/` folder
2. Reference in code: `/image-name.png`

## 🌐 Deployment

### Netlify (Recommended)
1. Build: `npm run build`
2. Drag `dist` folder to [Netlify Drop](https://app.netlify.com/drop)

### Vercel
1. Install: `npm i -g vercel`
2. Deploy: `vercel`

### GitHub Pages
1. Install: `npm install --save-dev gh-pages`
2. Add to `package.json`:
   ```json
   "homepage": "https://yourusername.github.io/repo-name",
   "scripts": {
     "deploy": "gh-pages -d dist"
   }
   ```
3. Deploy: `npm run deploy`
